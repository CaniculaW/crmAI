## 4. 环境限制

完整 Compose 生产包构建重试仍失败，失败点为 Docker Hub 匿名 token 请求超时：

```text
failed to fetch anonymous token: Get "https://auth.docker.io/token?...": dial tcp 66.220.148.145:443: i/o timeout
```

影响：

- `postgres:16` 已拉取并可用。
- `maven:3.9-eclipse-temurin-17`、`eclipse-temurin:17-jre`、`node:22-alpine`、`nginx:1.27-alpine` 未能拉取 metadata。
- 因此本轮未完成 `docker compose -f compose.v1-test.yml up -d --build` 的完整后端/前端镜像启动。

Browser 插件验证本地 URL 时曾触发安全策略阻断：

```text
Browser Use cannot visit the requested page because its URL is blocked by the Browser Use URL policy.
```

影响：

- 本轮已通过项目内置 Playwright 浏览器 Smoke 验证本地前端入口。
- 具名测试环境仍需在可访问目标环境的测试执行机上运行同一 Smoke 并归档截图：

```bash
cd frontend
CRM_SMOKE_URL=https://<test-frontend-domain>/system \
CRM_SMOKE_USERNAME=<test-admin-username> \
CRM_SMOKE_PASSWORD=<test-admin-password> \
CRM_SMOKE_EXPECTED_TEXTS="<组织名称>,<管理员姓名>,<角色编码>" \
npm run smoke:v1:browser
```
