# CRM V1 UAT 证据包与 Go/No-Go 记录模板

版本：V1.0

适用场景：具名测试环境完成 V1 试点验收时，由测试负责人汇总自动化结果、部署态 Smoke、业务演示、缺陷状态和准出结论。

填写原则：

- 不记录明文密码、生产密钥、个人敏感信息。
- 不虚构销售侧或管理侧验收人。
- 截图和命令输出需来自具名测试环境或明确标记为本地验证。
- 通过项、业务验收和签署证据需指向 `docs/` 下已归档文件或外部 `http(s)` 系统链接。
- P0 阻断缺陷未关闭时不得给出 Go 结论。
