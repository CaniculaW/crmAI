## 4. 默认工程实施边界

若启动会未提出替代方案，默认采用以下工程边界进入 Sprint 0 研发：

- 仓库形态：`code` 下单仓库管理前端、后端和项目级文档。

- 前端：React + TypeScript + Ant Design。
- 数据库：PostgreSQL，迁移工具优先 Flyway。
- 第一批代码目标：后端健康检查、基础模块结构、前端工程骨架、路由、请求层、基础布局。
