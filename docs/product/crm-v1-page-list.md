版本：V1.0

日期：2026-06-17

状态：Sprint 0 产品评审稿，字段与筛选项来自 PRD 和系统设计文档，待业务与研发确认后进入研发任务池。

## 1. 范围说明

V1 页面覆盖登录、工作台、客户池、联系人与干系人、商机、销售行动、周进展和系统管理。V1 不建设方案标书、合同、开票、回款、经营驾驶舱完整指标和 AI 销售助手完整能力。

## 3. 逐页清单

### 3.1 登录页

| 项目 | 内容 |
|---|---|
| 核心目标 | 支持用户通过账号密码进入 CRM，并加载角色、菜单和数据范围 |
| 主要组件 | 登录表单、账号输入、密码输入、登录按钮、忘记密码入口、错误提示 |
| 关键字段/筛选 | 账号、密码、验证码或二次校验为待确认 |
| 验收点 | 用户可登录；停用用户无法登录；登录失败有明确提示；登录成功后进入工作台 |

### 3.2 忘记密码页

| 项目 | 内容 |
|---|---|
| 核心目标 | 支持用户发起密码重置，或引导管理员重置 |
| 主要组件 | 账号输入、邮箱/手机校验占位、提交按钮、返回登录 |
| 关键字段/筛选 | 账号、邮箱/手机号；实际重置方式待确认 |
| 验收点 | 可提交重置申请；无效账号不泄露敏感信息；重置路径和提示清晰 |

### 3.3 修改密码页

| 项目 | 内容 |
|---|---|
| 核心目标 | 支持已登录用户修改密码 |
| 主要组件 | 旧密码、新密码、确认新密码、强度提示、保存按钮 |
| 关键字段/筛选 | 旧密码、新密码、确认新密码 |
| 验收点 | 修改密码需校验旧密码；新密码需满足强度规则；修改成功后可重新登录 |

### 3.4 我的待办

| 项目 | 内容 |
|---|---|
| 核心目标 | 汇总当前用户需要处理的行动、提醒和待补全事项 |
| 主要组件 | 待办列表、逾期标记、今日/本周分组、快捷完成、跳转详情 |
| 关键字段/筛选 | 待办类型、状态、截止时间、关联客户、关联商机、责任人 |
| 验收点 | 可看到本人待办；逾期事项高亮；可跳转到销售行动或商机详情 |

### 3.5 我的商机

| 项目 | 内容 |
|---|---|
| 核心目标 | 让销售快速查看本人负责或协同的进行中商机 |
| 主要组件 | 商机卡片/表格、阶段分布、风险标记、最近跟进摘要、新建商机按钮 |
| 关键字段/筛选 | 商机阶段、商机状态、风险状态、预计成交时间、负责人 |
| 验收点 | 默认不展示关闭或取消跟进商机；可按阶段和风险筛选；可进入商机详情 |

### 3.6 今日行动

| 项目 | 内容 |
|---|---|
| 核心目标 | 展示今日计划或发生的销售行动 |
| 主要组件 | 时间线、行动状态、行动类型、快捷完成、延期入口 |
| 关键字段/筛选 | 行动时间、行动状态、行动类型、关联客户、关联商机 |
| 验收点 | 今日行动按时间排序；可完成行动；完成后触发最近跟进回写 |

### 3.7 本周行动

| 项目 | 内容 |
|---|---|
| 核心目标 | 展示本周计划、完成、逾期和风险行动 |
| 主要组件 | 周视图、状态统计、行动列表、风险筛选 |
| 关键字段/筛选 | 自然周、行动状态、行动类型、风险类型、责任人 |
| 验收点 | 可按自然周查看；已完成且进入周进展的行动可被周进展汇总引用 |

### 3.8 客户列表

| 项目 | 内容 |
|---|---|
| 核心目标 | 支持销售和管理者检索、筛选、维护客户池 |
| 主要组件 | 客户表格、筛选栏、新建客户、批量操作占位、分页 |
| 关键字段/筛选 | 客户名称、客户类型、行业、区域、客户等级、客户状态、负责人、归属部门、最近跟进时间 |

### 3.9 客户详情

| 关键字段/筛选 | 客户名称、类型、行业、区域、等级、状态、负责人、协同人员、客户背景、重点需求方向、最近跟进 |
| 验收点 | 可看到联系人、商机、行动入口；完成销售行动后最近跟进时间和摘要自动更新；支持编辑客户 |
