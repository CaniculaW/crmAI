## 1. 候选版本定位

`v1.0.0-rc.1` 是进入具名测试环境部署和业务验收准备的第一个 V1 候选版本。候选版本 tag 指向包含本版本记录的提交；上方验证基线 commit 是 GitHub Actions 已完成远端绿灯验证的代码基线。

该版本已完成研发侧自动化验证、部署配置校验和本地部署态 Smoke 证据归档；仍需在具名测试环境完成 UAT 演示、缺陷闭环和销售侧/管理侧验收签署后，才能升级为正式 V1 发布版本。

## 2. V1范围

本候选版本覆盖：

- 登录、登出、修改密码、管理员重置密码。
- 组织、用户、角色、权限、数据范围和字典维护。
- 客户池、联系人与干系人、商机生命周期、销售行动、提醒待办、附件元数据。
- 销售行动完成后回写客户和商机最近跟进。
- 商机周进展按已完成销售行动自动汇总。
- 关键操作审计日志。
- V1演示管理员种子和部署态 bootstrap 探针。
- 前端 V1 基础页面、核心对象筛选、详情入口和主要编辑操作。

本候选版本不包含：

- 方案标书。
- 合同管理。
- 开票和回款管理。
- 经营驾驶舱完整指标。
- AI销售助手完整能力。
