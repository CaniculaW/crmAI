## 1. 候选版本定位

`v1.0.0-rc.5` 是在 `v1.0.0-rc.4` 基础上新增 UAT 证据包生成器的 V1 候选版本，用于继续推进具名测试环境 UAT 和 Go/No-Go 准出。

该版本包含：

- `v1.0.0-rc.4` 已具备的 RC/UAT readiness 审计、本地具名验证环境证据和证据版本一致性检查。
- `scripts/v1-uat-evidence-pack.mjs`：按具名测试环境名称、前后端地址、Git 提交号和候选版本生成 UAT 证据包草稿。
- `scripts/v1-uat-evidence-pack.test.mjs`：覆盖 UAT-001 至 UAT-010 生成、密码/API Token 不落文档、CLI 参数解析。
- GitHub Actions `V1 Validation` 增加 UAT 证据包生成器测试和 dry-run 生成。

该版本已完成研发侧自动化验证、部署配置校验、RC/UAT 就绪审计和本地服务级具名环境验证；仍需在项目指定具名测试环境完成 UI 级浏览器 Smoke、UAT 演示、缺陷闭环和销售侧/管理侧验收签署后，才能升级为正式 V1 发布版本。

## 2. V1范围

本候选版本覆盖：

- 登录、登出、修改密码、管理员重置密码。
- 组织、用户、角色、权限、数据范围和字典维护。
- 客户池、联系人与干系人、商机生命周期、销售行动、提醒待办、附件元数据。
- 销售行动完成后回写客户和商机最近跟进。
- 商机周进展按已完成销售行动自动汇总。
- 关键操作审计日志。
- V1演示管理员种子和部署态 bootstrap 探针。
- 前端 V1 基础页面、核心对象筛选、详情入口和主要编辑操作。
- UAT 证据包草稿生成器。

本候选版本不包含：

- 方案标书。
- 合同管理。
- 开票和回款管理。
- 经营驾驶舱完整指标。
- AI销售助手完整能力。

## 7. 已知限制

- 当前机器访问 Docker Hub token 接口曾超时，因此本地未完成 `docker compose -f compose.v1-test.yml up -d --build` 的完整后端/前端镜像启动；Compose 配置、后端 package、PostgreSQL迁移集成、前端 build 和本地服务级验证已分别通过。

- 业务验收人、具名测试环境域名、正式测试账号和业务样例数据仍待项目/测试/业务侧确认。
