# CRM V1.0.0-rc.6 候选版本记录

发布日期：2026-06-18

计划 Git tag：

```text
v1.0.0-rc.6
```

## 1. 候选版本定位

`v1.0.0-rc.6` 是在 `v1.0.0-rc.5` 基础上补强具名测试环境部署可靠性的 V1 候选版本，用于继续推进正式 UAT 和 Go/No-Go 准出。

该版本包含：

- `v1.0.0-rc.5` 已具备的 RC/UAT readiness 审计、本地具名验证环境证据、UAT 证据包生成器和远端 `V1 Validation` 质量门。
- `backend/Dockerfile` 和 `frontend/Dockerfile` 支持通过 build args 覆盖构建/运行基础镜像。
- `compose.v1-test.yml` 支持通过 `.env` 覆盖 `CRM_POSTGRES_IMAGE`、`CRM_BACKEND_BUILD_IMAGE`、`CRM_BACKEND_RUNTIME_IMAGE`、`CRM_FRONTEND_BUILD_IMAGE`、`CRM_FRONTEND_RUNTIME_IMAGE`。
- `scripts/v1-deployment-config-check.mjs`：验证 Dockerfile、Compose、`.env.example` 和部署手册已支持镜像源覆盖。
- `scripts/v1-deployment-config-check.test.mjs`：覆盖硬编码基础镜像和缺少镜像代理说明的失败场景。
- GitHub Actions `V1 Validation` 增加部署镜像覆盖配置检查和对应测试。

该版本已完成研发侧自动化验证、部署配置校验、RC/UAT 就绪审计和本地服务级具名环境验证；仍需在项目指定具名测试环境完成 UI 级浏览器 Smoke、UAT 演示、缺陷闭环和销售侧/管理侧验收签署后，才能升级为正式 V1 发布版本。

## 2. V1范围

本候选版本覆盖：

- 登录、登出、修改密码、管理员重置密码。
- 组织、用户、角色、权限、数据范围和字典维护。
- 客户池、联系人与干系人、商机生命周期、销售行动、提醒待办、附件元数据。
- 销售行动完成后回写客户和商机最近跟进。
- 商机周进展按已完成销售行动自动汇总。
- 关键操作审计日志。
- V1演示管理员种子和部署态 bootstrap 探针。
- 前端 V1 基础页面、核心对象筛选、详情入口和主要编辑操作。
- UAT 证据包草稿生成器。
- 测试环境镜像源覆盖配置。

本候选版本不包含：

- 方案标书。
- 合同管理。
- 开票和回款管理。
- 经营驾驶舱完整指标。
- AI销售助手完整能力。

## 5. 镜像源覆盖

默认 `.env` 镜像配置：

```text
CRM_POSTGRES_IMAGE=postgres:16
CRM_BACKEND_BUILD_IMAGE=maven:3.9-eclipse-temurin-17
CRM_BACKEND_RUNTIME_IMAGE=eclipse-temurin:17-jre
CRM_FRONTEND_BUILD_IMAGE=node:22-alpine
CRM_FRONTEND_RUNTIME_IMAGE=nginx:1.27-alpine
```

若测试机访问 Docker Hub token 接口超时，可替换为企业镜像代理或内网镜像仓库地址。部署说明见：

```text
docs/deployment/v1-test-environment-compose.md
```

## 6. UAT证据包生成

示例命令：

```bash
node scripts/v1-uat-evidence-pack.mjs \
  --environment <test-env-name> \
  --frontend-url <test-frontend-url> \
  --backend-url <test-backend-url> \
  --git-commit <git-sha> \
  --rc v1.0.0-rc.6 \
  --username <masked-admin-user> \
  > crm-v1-uat-evidence-pack.md
```

生成器只生成可填写的证据包草稿，不替代真实测试执行、截图归档、缺陷闭环和业务签署。
